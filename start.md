# 多媒体教学Wiki



## 这个Wiki是用来干什么的？

本Wiki旨在帮助一线教师和一线电教委员（学生）更好地解决一线教学过程中产生的各种多媒体电脑使用问题，并提供一些电脑使用提示及使用逻辑教学，具有较强的逻辑性、实践性和参考性，适合广大师生群体使用。

## 声明
本Wiki无盈利，仅作为教学设备使用说明平台及教学问题求助平台。本Wiki中出现的任何观点均不代表Wiki官方观点，仅供广大师生交流参考。尽管本Wiki中所有解决方案均已在部分学校中经过实践检验有效，但并不排除部分解决方案存在错误或过时的可能，在此恳请广大师生不吝斧正，使本Wiki内容变得更加科学完善。

本Wiki仅以**日常使用**为说明目的，对于部分软件的完全使用方法，本Wiki不作赘述。如果你想进一步深入了解本Wiki提到的软件，请查阅软件发布商所提供的软件使用帮助。

有关创作者对本Wiki上的内容享有充分的知识产权。
